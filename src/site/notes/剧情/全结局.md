## 1结局（1f）
{ #1end}


```
你四脚朝天地瘫在地板上。
远处传来干员们交谈的声音，混着脚步、文件夹翻动、咖啡杯磕在桌沿的轻响。
空气里浮着一点淡淡的中央空调清新剂的味道，闻起来像被反复打扫过的清晨。
你身侧的地板已经积了一层薄灰，该擦一擦了。
“管理员？”
是佩丽卡，监督的、官方发言人的、通讯频道里那道沉稳的声音。
急促的脚步声从另一边跑过来，在你身边猛地刹住，一道影子俯下来挡在你上方。
佩丽卡想要伸手来扶你，神情中的慌乱没有半分遮掩。
“——管理员，你看着我，看着我，能听见吗？”
…………
你看着慌乱的她——你先前认识的那位监督是她，你在那片空间认识到的也是她。
而现在你所看到的，或许才是完整的她，真正的她。
你第一次觉得，有些话，或许该由你先开口了。
```

## 2结局（2f）
{ #2end}


```
你已经在这里转了多久？
深入、深入、深入……没有尽头。
直到刺目的金黄色电流撕裂了前方的墙壁，光从裂缝里透露出来，越来越亮。
——佩丽卡终于找到了你。
她没有说话，只是走过来，极为用力地攥住了你的手腕
，你就这样被她拖拽着，离开了这片荒谬的协议空间，跌回了现实?
她依旧一言不发。
是她还是"她？"
…………
有几面墙飞了过来，带着令人窒息的风压
…………
这里一直有第三人吗？
…………
刺目的阳光让我睁不开眼
“生产千万条，安全第一条，四号谷地提醒您……”
为什么……我在四号谷地？
```


## 3结局（大于2f）
{ #3end}


```
内化宇宙确实是一片神奇且美丽的空间，你也这么觉得吧，管理员。
——或者说，博士、预言家？
呵，你还真是一点都没变
明明沿着熵增的方向走了这么远，却还是……
学不会该如何跟那些……心思太重的姑娘们相处呢。
我懂的。你想用这种方式来走进她们的内心
去看看她们到底在想什么、怕什么、把什么东西藏得这么深……
只是，除我之外，还没有谁会在内心深处的迷宫里为你留出一条专属通道呢。
而你也总是这样，一头扎进去，就忘了家的方向。
真是拿你没办法。不过，也没有关系。
在你继续去追逐那份属于你我的愿景之前，就先在这儿好好休息一下吧。
我来帮你处理这个问题。
```
